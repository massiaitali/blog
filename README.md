# Blog

A static blog built with [Nuxt Content](https://content.nuxt.com), deployed via SSG on Vercel.

## Setup

Install dependencies:

```bash
vp install
```

## Development

```bash
vp run dev
```

## SSG Build

Generate the static site:

```bash
vp run generate
```

Preview the static output locally:

```bash
vp run preview
```
